🐳 Docker Commands

Run these commands in your terminal:

# Start the isolated import container
docker-compose -f docker-compose.turbo-import.yml up --build

# View real-time logs
docker logs garage-turbo-import -f

# Check container status
docker ps | grep turbo-import

# Stop the container
docker-compose -f docker-compose.turbo-import.yml down

# View container resource usage
docker stats garage-turbo-import
